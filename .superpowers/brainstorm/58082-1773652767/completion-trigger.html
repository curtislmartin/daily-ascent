When should the nudge disappear?

A

Any field filled in

Badge and banner disappear as soon as the user adds at least one demographic (e.g. just age range).

B

All four fields filled in

Nudge persists until age range, sex, height range, and activity level are all set.

C

Explicit "Save" tap

A demographicsCompleted flag on UserSettings. Set when the user taps Save — regardless of how many fields they filled in. Clearest signal of intent.
